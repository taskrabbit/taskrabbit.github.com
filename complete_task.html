Completing a Task

TaskRabbit allows completing a Task

Setup

  • There is a Rabbit who is authenticated to your client application with id "jhfe92kjfklep902mrdfg3jdsuo32b1os"
  • The client application secret is "fei3on2o1in321XOIENOIn2io13zoin21x"
  • There is an opened task "My Task" with id 1452
  • The user has an authorization token of "rRou72pewt102y926jfhwow438238poQR31d"
  • The user is the Rabbit of the task "My Task"

POST /api/v1/tasks/1452/complete

  • Authorization: OAuth rRou72pewt102y926jfhwow438238poQR31d

Simplified Response

  • Status Code: 200
  • Content-Type: application/json
{
  "id":1452,
  "name":"My Task",
  "state":"completed"
}

Error Cases

The server will respond with different status codes when there is a problem completing the Task.

Incorrect Authorization

POST /api/v1/tasks/1452/complete

  • Authorization: OAuth XXXXXX
  • X-Client-Application: fei3on2o1in321XOIENOIn2io13zoin21x

401 Unauthorized

  • Content-Type: application/json
{
  "error":"There must be an authenticated user for this action"
}
