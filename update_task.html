Updating a Task

TaskRabbit allows updating an opened Task

Setup

  • There is a user who is authenticated to your client application with id "jhfe92kjfklep902mrdfg3jdsuo32b1os"
  • The client application secret is "fei3on2o1in321XOIENOIn2io13zoin21x"
  • The user has an authorization token of "rRou72pewt102y926jfhwow438238poQR31d"
  • The user has an opened task "My Task" with id 1452

PUT /api/v1/tasks/1452

  • Authorization: OAuth rRou72pewt102y926jfhwow438238poQR31d
  • Content-Type: application/json
{
  "name":"My New Task"
}

Simplified Response

  • Status Code: 200
  • Content-Type: application/json
{
  "id":1452,
  "name":"My New Task",
  "state":"opened"
}

Error Cases

The server will respond with different status codes when there is a problem updating the Task.

{

Incorrect Authorization

  • Authorization: OAuth XXXXXX
  • X-Client-Application: fei3on2o1in321XOIENOIn2io13zoin21x
{
  "name":"My Other Task"

{
  "error":"There must be an authenticated user for this action"
